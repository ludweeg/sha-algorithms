# SHA-algorithms
The five secure hash algorithms: SHA-1, SHA-224, SHA-256, SHA-384, and SHA-512.

Information (bits):

| Algorithm  |  Message Size  | Word Size  | Message Digest Size |
| :--------: | :------------: | :--------: | :-----------------: |
| sha1 | < 2^64 | 32 | 160 |
| sha224 | < 2^64 | 32 | 224 |
| sha256 | < 2^64 | 32 | 256 |
| sha384 | < 2^128 | 64 | 384 |
| sha512 | < 2^128 | 64 | 512 |
